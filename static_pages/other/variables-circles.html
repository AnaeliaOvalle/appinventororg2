What is a Variable?

A variable labels (or names) a piece of text, a list, or a number and can be changed in different places and at different times.
Like in the video's example, the variable Tips stood for however much has been made in tips and he uses the tips variable to find out how much was made during a particular hour.


In coding, you define a variable when you need to remember a number, text, and/or when there isn't a component property for storing the information you need. A great example of this comes from making the PaintPot app, when you need to control the size of a circle drawn by the user.

Challenge

Make it so that each time the user clicks a circle you double the previous' circle size.

Hint:

Currently, the size of the circle is dependent on the variable global circleSize.

In order to manipulate the size of the circle, we need to change the variable “circleSize”
We need circleSize to double each time the canvas is touched.

Here, The circle is drawn with the radius circleSize. Then the variable circleSize is doubled (multiplied by 2) so that the next time a circle is clicked it is double the size.
