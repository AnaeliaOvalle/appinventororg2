Java Bridge Programming

The App Inventor Java Bridge is a Java library for building Android apps. It's much easier to learn than the Android SDK and it uses the same simple terminology as App Inventor. If you are an App Inventor programmer and want to transition to Java, the App Inventor Java Bridge is for you.

Setup

To start building apps with the Java Bridge, simply import the Java Bridge jar file into your Android project.

Sample Apps

ClickRed -- A simple app to demonstrate the form of a Java Bridge app, click a button to turn it red. PaintPot -- Draw on a picture you take with the camera! This app introduces the Canvas component, drawing commands, and variables. MoleMash -- Click the mole to earn points! This app introduces Canvas, ImageSprite and Clocks timers. President's Quiz -- A trivia app about US Presidents, you can customize to make your own quizzes. Learn about lists and how to index through a list. Stock Market App -- Track your favorite stocks in real-time. Learn about the Web component and APIs.
