Introduction to App Inventor Java Bridge Programming

The App Inventor Java Bridge is a Java library for building Android apps. It has a much easier learning curve than the Android SDK, which is the library commonly used in Android development. The Java Bridge has an added advantage for those who have programmed in App Inventor, as the terminology is the same: there is a Java class pertaining to each App Inventor component, and the methods of each class correspond to App Inventor component properties or functions. If you are an App Inventor programmer and want to transition to Java, the App Inventor Java Bridge is for you.

Sample Java Bridge Apps

To program with the Java Bridge, import this library jar file into your Android project (in either Eclipse or Android Studio).

Here are some sample apps (Java code):

  • ClickRed -- A simple app to demonstrate the form of a Java Bridge app, click a button to turn it red.
  • PaintPot -- Draw on a picture you take with the camera! This app introduces the Canvas component, drawing commands, and variables.
  • MoleMash -- Click the mole to earn points! This app introduces Canvas, ImageSprite and Clocks timers.
  • President's Quiz -- A trivia app about US Presidents, you can customize to make your own quizzes. Learn about lists and how to index through a list.
  • Stock Market App -- Track your favorite stocks in real-time. Learn about the Web component and APIs.
