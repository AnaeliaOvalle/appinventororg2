Example 2. How do you say "I’m shaking” when the user shakes the phone?

This app has two components, an AccelerometerSensor and a TextToSpeech. When the user shakes the phone or tablet, the AccelerometerSensor senses the event and the when AccelerometerSensor1.Shaking event handler is triggered. The TextToSpeech component is called to carry out the response of speaking the words “I’m shaking”.
