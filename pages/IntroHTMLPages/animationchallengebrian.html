When the GoButton is clicked do the following by using the Clock Component (You should only be using the clock component to move the balls)

  1. Have one ball move right and when it hits the edge - move left then hits the edge - move right continuously
  2. Have another ball move up and when it hits the edge - move down then hits the edge - move down continuously.
  3. Have another ball move diagonally and when it hits the edge move diagonally in the opposite direction continuously.
