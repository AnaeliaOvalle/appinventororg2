Learning Objectives

  • Understand how the Canvas Component works and the different function calls it has.
  • Understand what variables are, how to create them, and where to use them in building apps.
  • Create a Paint Pot style app where users can draw on a Canvas.
  • Understand how to add and subtract numbers.
  • Learn how to use the Clock Component for timers and to move the mole in Mole Mash.
  • Understand how animation works and how to animate sprites by using both the Clock and internal sprite propertiesS.
  • Create Pong!
  • Create your own game!!!
