In this section you will learn about the Canvas (a component that allows you to draw and animate sprites), animation, and game making. With the Canvas Component, you will be building a painting app, molemash style arcade game, learn about animation, and build your own pong app.

Learning Objectives

  • Understand how the canvas component works and the different function calls it has.
  • Understand what variables are, how to create them, and where to use them in building apps.
  • Create a Paint Pot style app where users can draw on a canvas.
  • Understand how to add up and subtract numbers.
  • Learn how to use the Clock component for timers and to move the mole in Mole Mash.
  • Understand how animation works and how to animate sprites by using both the Clock and internal sprite components.
  • Create Pong!
  • Create your own game!!!
