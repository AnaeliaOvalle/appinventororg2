How does an app remember things?

Memory

An app remembers things, it has "hidden" memory. Think of it like a spreadsheet of data in the app's private "brain". You, the programmer, control the memory - what should be remembered and when to remember it. When you drag a component (e.g., button, text box) into your app, a set of properties are remembered, by being named, and stored. These properties are "spreadsheet" cells that you can change.

For Example

Look at the properties for a Button1. Each property is named, has a value, and we can ask the app to tell us each property's value.

Enabled is set to True
FontBold is set to False
FontSize is set to 14
Text is set to "Text for Button1"
